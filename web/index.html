How to Use StoryForge

How does the pipeline work?

Idea → Layer 1: Story Generation → Layer 2: Simulation

Layer 1: Story Generation

  • Create characters with personality, backstory, and motivation
  • Build world context and setting
  • Generate detailed outline + write each chapter automatically

Layer 2: Drama Simulation

  • Each character becomes an autonomous AI agent
  • Agents interact, confront, and betray each other
  • Rewrites the story with a higher drama score

How to Use

  1. Go to Settings → enter API Key and choose a model
  2. Go to Create Story → enter your idea
  3. Adjust parameters if needed
  4. Click Run Pipeline and wait
  5. View results, export files, read the story

Compatible APIs

Supports: OpenAI, DeepSeek, Gemini, Groq, Together AI, OpenRouter, Ollama, and any provider with a /v1/chat/completions endpoint
