# Product Hunt Launch Checklist — StoryForge

## Pre-Launch (Complete ≥ 3 days before)

- [ ] Landing page live and mobile-responsive (`landing/index.html`)
- [ ] Sample story gallery published (`landing/gallery.html`)
- [ ] GitHub repo has 5+ stars and a clean README
- [ ] Demo GIF or 60-second screen recording ready
- [ ] All social profiles updated (Twitter/X, LinkedIn) with StoryForge link
- [ ] Maker account created on Product Hunt and connected to GitHub
- [ ] "Notify me on launch" teaser posted to gather early upvotes

## Launch Day Assets

- [ ] **Tagline** (≤ 60 chars): "Turn one sentence into a full AI-generated story"
- [ ] **Description** (≤ 260 chars): highlights self-hosted, Vietnamese-first, 3-layer pipeline
- [ ] **Thumbnail** 240×240 px — StoryForge logo on dark background
- [ ] **Gallery screenshots** (3 minimum):
  - [ ] Screenshot 1 — generation UI / pipeline progress
  - [ ] Screenshot 2 — sample story web reader
  - [ ] Screenshot 3 — storyboard / export formats
- [ ] **Maker comment** posted within first 15 minutes of launch explaining the backstory

## Launch Day Actions

- [ ] Post at **Tuesday 12:01 AM PT** for maximum 24-hour window
- [ ] Share on Twitter/X, LinkedIn, relevant subreddits (r/MachineLearning, r/selfhosted)
- [ ] Post in Discord `#announcements` channel
- [ ] Notify contributors and ask for upvotes (no vote-trading)

## Post-Launch

- [ ] Respond to every Product Hunt comment within 2 hours on launch day
- [ ] Thank top commenters publicly
- [ ] Write a "launch retrospective" post for GitHub Discussions
- [ ] Update README with Product Hunt badge if ranked top 5

## Timing Recommendation

Launch on a **Tuesday at 12:01 AM Pacific Time**.  Tuesdays historically see the
highest hunter traffic; the 12:01 AM slot gives the full 24-hour voting window.
